Objetivo

La meta principal de mi aplicación web es optimizar el control del tráfico en nuestra ciudad. Ofrezco a los ciudadanos y autoridades una herramienta eficiente y accesible para monitorear el tráfico, gestionar eventos viales y fomentar una circulación más fluida y segura en nuestras calles.

Tecnologías

Estoy aprovechando una combinación de tecnologías líderes para dar vida a mí proyecto.

Inicié con HTML, la estructura fundamental de mis páginas web, permitiéndome organizar y presentar información de manera efectiva en este proyecto en el que trabajo solo..

Utilizando CSS, he dado vida a las páginas al diseñar estilos atractivos y coherentes, mejorando así la experiencia visual de los usuarios en este proyecto en el que estoy trabajando.

Para agilizar el diseño y garantizar la consistencia, estoy aprovechando Bootstrap, una biblioteca de diseño receptivo que me ayuda a crear interfaces modernas y adaptables en diferentes dispositivos.

Angular desempeña un papel fundamental en mi conjunto de herramientas, permitiéndome construir una aplicación web dinámica y de una sola página. Gracias a su capacidad para manejar interacciones en tiempo real y su estructura modular, estoy creando una experiencia interactiva fluida para los usuarios.

La API desarrollada en C# es la columna vertebral de la aplicación, permitiendo la comunicación fluida entre el frontend y el backend. Esto garantiza un manejo eficiente de los datos y una interacción dinámica.

En cuanto al control de versiones, estoy utilizando Git. Esta tecnología me permite gestionar las versiones del proyecto, reduciendo los riesgos de pérdida de código. Proporciona un historial completo de todas las iteraciones, rastrea los cambios realizados y me permite revertirlos si es necesario, además de facilitar la implementación de mejoras en el proyecto.

Para el almacenamiento y la gestión de datos, confiamos en SQL Server. Esta base de datos robusta me proporciona un entorno seguro y eficiente para almacenar y recuperar datos importantes, lo que garantiza que la aplicación esté respaldada por información confiable y actualizada.
